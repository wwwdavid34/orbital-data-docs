# Attribution

## Data attribution

All satellite orbital data served by the Orbital Data API originates from the **United States Space Command (USSPACECOM)** via **Space-Track.org**. Per Space-Track's terms of use, you must acknowledge the original data source in any application or publication that uses this data.

### Required citation

Include the following attribution wherever you display or publish data obtained from this API:

!!! important "Required"
    **Source: USSPACECOM via Space-Track.org**

This applies to all formats (JSON, TLE, XML, etc.) and all data types (GP element sets, SATCAT metadata).

## Powered by Orbital Data API

If you build an application or service that uses the Orbital Data API, we ask that you include a "Powered by" attribution. This is not legally required, but it helps others discover the API and supports the community.

### Text attribution

> Powered by [Orbital Data API](https://docs.orbitaldata.dev)

### Suggested placement

- Footer of web applications
- About/credits page
- README of open-source projects
- Data source section of research papers
